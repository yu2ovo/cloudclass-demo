Project Notes

          
该 Demo 和原项目的关系

          
完整云学堂在线平台项目依赖多个中间件，免费平台不适合全量部署。本 Demo 将核心业务流程前端化，适合放到简历作为可访问演示入口。

        

        

          

            
完整项目

            
Spring Cloud Alibaba、Nacos、Gateway、OpenFeign、MySQL、Redis、RabbitMQ、Elasticsearch、MinIO、XXL-JOB、OAuth2/JWT。

          

          

            
本 Demo

            
纯 HTML/CSS/JavaScript 静态页面，不连接真实数据库和中间件，主要用于展示业务流程、技术理解和项目差异化能力。
